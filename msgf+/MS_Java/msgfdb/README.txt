Requirements
	- JRE 1.6 or greater
	- Main memory 2GB or greater (recommended 4GB)

Installation
	- Unzip MSGFPlus.zip
	- Place MSGFPlus.jar in any folder

Usage Information
	- Type 'java -jar MSGFPlus.jar' for command line arguments
	- For detailed documentation, visit https://bix-lab.ucsd.edu/pages/viewpage.action?pageId=13533355
